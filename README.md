# react-starter

[![Dependency Status][depstat-image]][depstat-url] [![devDependency Status][dev-depstat-image]][dev-depstat-url]

> React Starter

## Install

```sh
$ git clone https://github.com/exeto/react-starter.git
$ cd react-starter && yarn install
```

## Usage

```bash
# Development
$ yarn start

# Build
$ yarn build
```
